프로그램 이름 : 자바 JAVA!
개요 : 게시판을 이용한  자바 프로그래밍에 대해 질문, 답변 및 토의할 수 있는 회원제 프로그램.
  관리자는 게시물 관리와 계정 조회 및 업데이트 할 수 있으며,사용자는 질의응답게시판 외에 쪽지와 실시간 채팅을 통해 정보 공유 가능

개발환경
-프로그래밍 언어 : JAVA 1.8
-데이터베이스 : Oracle11g express edition
-운영체제 : Window 10
-IDE : eclipse Version 2019-06 (4.12.0)

맡은 기능
-건의게시판 사용자
-건의게시판 관리자
